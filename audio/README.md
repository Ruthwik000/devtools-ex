# Audio Files for ExPro Extension

## Alert Sound

Place your alert sound file here as `alert.mp3` or `alert.wav`

The alert sound will play when:
- Focus Detection detects a phone in the webcam feed

### Recommended Audio Specifications:
- Format: MP3 or WAV
- Duration: 1-3 seconds
- Volume: Moderate (will be played at 70% volume)
- Type: Alert/notification sound

### Free Sound Resources:
- https://freesound.org/
- https://mixkit.co/free-sound-effects/
- https://www.zapsplat.com/

### Example Files to Use:
- `alert.mp3` - Main alert sound for phone detection
- `alert.wav` - Alternative format

**Note:** Make sure to add the audio file to this folder and update the manifest.json to include it in web_accessible_resources.
