SINGLE PURPOSE DESCRIPTION
================================================================================

DevTools provides essential developer utilities including SEO analysis, cookie management, font inspection, color picking, and AI-powered code assistance for GitHub repositories and web page content analysis.

================================================================================
PERMISSION JUSTIFICATIONS
================================================================================

1. STORAGE JUSTIFICATION:
--------------------------
Required to save user preferences, toggle states, API keys for AI features, and tab group data for the Tab Manager functionality.


2. TABS JUSTIFICATION:
--------------------------
Required to access current tab information for developer tools features, manage tab groups in Tab Manager, and inject content scripts for SEO checker, cookie editor, and other utilities.


3. BROWSINGDATA JUSTIFICATION:
--------------------------
Required for the Auto Clear Cache feature which allows developers to clear browser cache on demand for testing purposes.


4. COOKIES JUSTIFICATION:
--------------------------
Required for the Cookie Editor tool which allows developers to view, edit, create, and delete cookies for testing and debugging web applications.


5. DECLARATIVENETREQUEST JUSTIFICATION:
--------------------------
Required for optional ad blocking functionality to improve page load times during development and testing.


6. DECLARATIVENETREQUESTWITHHOSTACCESS JUSTIFICATION:
--------------------------
Required to apply network request filtering rules across all websites for the ad blocking feature.


7. SCRIPTING JUSTIFICATION:
--------------------------
Required to inject content scripts that provide developer tools UI overlays including SEO checker, font finder, color picker, and cookie editor on web pages.


8. ACTIVETAB JUSTIFICATION:
--------------------------
Required to access the current active tab's content for analyzing SEO metrics, extracting colors and fonts, and providing AI-powered assistance.


9. HOST PERMISSION JUSTIFICATION (<all_urls>):
--------------------------
Required to provide developer tools functionality across all websites. The extension needs to inject UI overlays for SEO analysis, cookie editing, font inspection, and color picking on any website the developer is working on.
